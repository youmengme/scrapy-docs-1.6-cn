Scrapy 文档快速入门指南¶

此文件提供了有关如何编译Scrapy文档的快速指南。

设置环境¶

要编译文档，您需要Sphinx Python库。要安装它及其所有依赖项，请从此目录运行以下命令

pip install -r requirements.txt

编译文档¶

要编译文档（到经典HTML输出），请从此目录运行以下命令:

make html

文档将生成(以HTML格式)在 build/html 目录中。

查看文档¶

要查看文档，请运行以下命令:

make htmlview

此命令将启动默认浏览器并打开（先前生成的）HTML文档的主页面。

重新生成¶

要清理所有生成的文档文件并从头开始运行:

make clean

请记住，此命令不会触及任何文档源文件。

即时重新创建文档¶

有一种方法可以在进行更改时自动重新创建文档，需要安装 watchdog(pip install watchdog)然后使用:

make watch
